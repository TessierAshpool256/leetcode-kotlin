## 6. Zigzag Conversion
**[⇐](../README.md)** |
<span style="color:orange">Medium</span> |

---
### En
The string `"PAYPALISHIRING"` is written in a zigzag pattern on a given number of rows like this: 
(you may want to display this pattern in a fixed font for better legibility)

```
P   A   H   N
A P L S I I G
Y   I   R
```

And then read line by line: `"PAHNAPLSIIGYIR"`


---
### Ru
Строка `"PAYPALISHIRING"` записывается в виде зигзагообразного узора по заданному количеству 
строк, например так:
```
P   A   H   N
A P L S I I G
Y   I   R
```

Затем её читают построчно слева направо, получая строку `"PAHNAPLSIIGYIR"`.
