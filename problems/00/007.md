## 7. Reverse Integer
**[⇐](../README.md)** |
<span style="color:orange">Medium</span> |

---
### En
Given a signed 32-bit integer `x`, return `x` with its digits reversed. If reversing `x` causes 
the value to go outside the signed 32-bit integer range `[-2³¹,2³¹ - 1]`, then return `0`.

Assume the environment does not allow you to store 64-bit integers (signed or unsigned).

---
### Ru
Дано 32‑разрядное знаковое целое число `x`. Требуется вернуть число, полученное из `x` путём 
обратного порядка его цифр. Если при обратном порядке цифр значение выходит за пределы диапазона знаковых 32‑разрядных целых чисел `[-2³¹,2³¹ - 1]`, следует вернуть `0`.

Дополнительное условие: в среде выполнения нельзя использовать 64‑разрядные целые числа (как знаковые, так и беззнаковые).
